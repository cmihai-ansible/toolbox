    Using the role

    Use a role to:

    You can create a role under roles/nvidia or download crivetimihai.nvidia using ansible-galaxy. Create a playbook called users-playbook.yml that defines / overrides the default variables of the role.

    Using the role in a playbook
    ---
- name: Install nvidia
  hosts: all

  tasks:
    - name: nvidia is configured
      import_role:
        name: crivetimihai.nvidia
      vars:
        nvidia_enable_service: true
      tags: nvidia

    # Install the role
ansible-galaxy install crivetimihai.nvidia

# Run the playbook
ansible-playbook nvidia-playbook.yml
